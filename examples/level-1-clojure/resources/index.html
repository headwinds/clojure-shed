Although Lakeside Lodge has the Retired Product tag, it can still be purchased on amazon but this is a warning sign to snap it up quick before its gone.

Some lego collectors see this Retired Product tag and think profit. There is aftermarket value for rare and discontinued lego sets. Since these sets are comprised of mainly common pieces, they can be re-created by hunting for and assembling the pieces by simply staring at the box art. Builders can also fairly easily search for the instructions. But it could be even easier still if the file for each set carried everything a lego enthusiast desired.

If lego was on a blockchain, it could provide a transaction history and carry all the blueprints. As the lego set passed from owner to owner, we would have an open record of its current state and it's histroy.

If the lego set was ever opened, that state should be recorded on the block and it would be de-valued losing it's prized "Unopened" or "Mint Condition" status. But who is responsible for recording this act on the chain?

I'll be using experiments like this to learn more about Clojure & ClojureScript as well as investigate how trust works on a blockchain involving real world goods and personal digital records.
